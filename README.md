# 🤡 TIW: Today I Wasted

> *"Why do it in 5 minutes when I can spend 4 hours automating it?"*

Welcome to the digital graveyard of my productivity. While the overachievers are filling up their **TIL (Today I Learned)** repos with brain wrinkles, I am here documenting the exact moments my last remaining brain cells evaporated. 

If you are looking for efficient code, leave. If you are looking for emotional damage, welcome.

---

## 📂 The Waste Basket (What's Actually Inside)

Despite the name, this is a chaotic archive of everything I do. If I spent my limited human lifespan staring at a screen to make it happen, it counts as a waste of time and goes right here.

### 🧮 1. Math & Algorithm Struggles (`/algorithms`)
*   Coding Problem solutions that probably violate every clean code principle known to mankind.
*   Writing horrific $O(N^{N^N})$ brute-force loops and apologizing to my CPU.
*   Solving math problems with code because my brain forgot how to do basic arithmetic.

### 🐛 2. The Bug Graveyard (`/bug-fixes`)
*   Documenting the miracle of code that worked perfectly yesterday but broke completely today for no logical reason.
*   Spending 5 hours scrolling through GitHub Issues only to realize the fix was a missing comma.
*   Gaslighting myself into thinking I am a genius after fixing a bug that I created in the first place.

### 📚 3. Shameless Self-Education (`/til`)
*   Standard TIL notes on things that everyone else figured out 5 years ago.
*   Messy summaries of official documentations I read right before passing out from boredom.
*   Random tech stacks I tried learning for 2 hours before completely abandoning them.

## 📜 Repository Guidelines

1.  **Zero Organization:** If creating folders becomes too tedious, files will be dumped straight into the root directory.
2.  **No Polishing:** No fancy formatting or sugarcoating. What you see is the raw, unedited panic of a desperate developer.
3.  **Inconsistency is Key:** Commit whenever I feel like it. True time-wasting follows no schedule.
