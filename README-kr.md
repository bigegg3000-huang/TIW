# 🗑️ TIW: Today I Wasted (오늘 내가 낭비한 것들)

🇺🇸 [English](README.md) | 🇰🇷 [한국어](README-kr.md)


> *"5분이면 끝날 일을 왜 4시간 동안 자동화하는거임?"*

제 생산성의 디지털 공동묘지에 오신 것을 환영합니다. 남부럽지 않게 갓생을 사는 사람들이 **TIL (Today I Learned)** 저장소에 뇌 주름을 채워 넣을 때, 저는 제 마지막 남은 뇌세포들이 증발하는 순간 순간들을 여기에 기록합니다.

효율적인 코드를 찾으러 오셨다면 그게 누구든 나가주세요. 정신적 공격을 받으러 오셨다면, 환영합니다.

---

## 📂 쓰레기통 (실제로 올라오는 것들)

여기는 제가 하는 모든 짓거리가 모이는 혼돈의 아카이브입니다. 제가 모니터를 쳐다보며 유한한 인간의 수명을 갈아 넣었다면, 그게 무엇이든 '시간 낭비'로 간주되어 바로 여기에 처박힙니다.

### 🧮 1. 수학 및 알고리즘과의 사투 (`/algorithms`)
*   인류에게 알려진 모든 클린 코드 원칙을 위배할 확률이 높은 코딩 문제 풀이
*   공포의 $O(N^{N^N})$ 짜리 무지성 반복문을 짜놓고 내 CPU에게 사죄하기.
*   내 뇌가 사칙연산을 까먹어 버리는 바람에 굳이 코드로 풀고 앉아있는 수학 문제들.

### 🐛 2. 버그 공동묘지 (`/bug-fixes`)
*   어제까지는 분명 완벽하게 돌아갔으나, 오늘 아무런 논리적 이유 없이 완전히 터져버리는 코드의 기적을 기록.
*   깃허브 이슈(GitHub Issues)를 5시간 동안 뒤진 끝에 해결책이 '쉼표 하나 누락'이었음을 깨닫게 되는 허무함.
*   결국 내가 만든 버그를 내가 고쳐놓고, 스스로 천재가 아닐까 착각하며 가스라이팅하기.

### 📚 3. 뻔뻔한 독학 흔적 (`/til`)
*   남들은 이미 5년 전에 다 깨우친 것들을 이제야 뒷북치며 적어두는 평범한 TIL 메모.
*   지루함에 기절하기 직전에 대충 끄적여 둔 공식 문서 요약본.
*   일단 배워보겠다고 까불다가 2시간 만에 완벽하게 유기해 버린 기술 스택들.

1.  **정리 정돈 안 함:** 폴더 만드는 것조차 귀찮아지면, 모든 파일은 그냥 루트(최상위) 디렉토리에 마구잡이로 던져집니다.
2.  **포장 금지:** 화려한 서식이나 미화는 없습니다. 여러분이 보시는 것은 벼랑 끝에 몰린 개발자의 날것 그대로의 패닉입니다.
3.  **불성실함이 핵심:** 내키는 대로 커밋합니다. 진정한 시간 낭비는 원래 계획을 따르지 않는 법입니다.
