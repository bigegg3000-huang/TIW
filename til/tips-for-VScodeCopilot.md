# VScode 꿀팁들
## 단축키
Ctrl + Shift + P = Open Command Palette (위에 명령어창)

Ctrl + ` = Open/Close Bottom Terminal (아래 터미널창)

Ctrl + B = Open/Close Left Sidebar (왼쪽 파일창)

Ctrl + Shift + I = Open Chatting Right Sidebar (오른쪽 채팅창)

## 코파일럿 요금제

https://docs.github.com/en/copilot/concepts/billing/usage-based-billing-for-individuals

Free는 딱 50번 질문 가능함.


| Plan | Price per month | Base credits | Flex allotment | Total monthly AI credits |
| :--- | :---: | :---: | :---: | :---: |
| **Free** | Free | 0 | 0 | 0 |
| **Student** | $2 USD | 200 | 0 | 200 |
| **Copilot Pro** | $10 USD | 1,000 | 500 | 1,500 |
| **Copilot Pro+** | $39 USD | 3,900 | 3,100 | 7,000 |
| **Copilot Max** | $100 USD | 10,000 | 10,000 | 20,000 |

1 AI 토큰은 0.01달러이며 모델별 요금제는 https://docs.github.com/en/copilot/reference/copilot-billing/models-and-pricing 이걸 참고

언제 어떤 모델을 쓸 지는 https://docs.github.com/en/copilot/tutorials/compare-ai-models 이거나 https://docs.github.com/en/copilot/reference/ai-models/model-comparison 이걸 참고
